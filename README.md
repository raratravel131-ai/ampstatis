# Rara Travel & Tour - AMP Static Site

Website statis AMP untuk perusahaan transportasi Rara Travel & Tour.

## Deployment
Deployed on Cloudflare Pages.

## Struktur File
- `index.html` - Halaman utama
- `_headers` - Custom HTTP headers
- `_redirects` - URL redirects (opsional)
- `assets/` - Gambar dan file statis (opsional)
